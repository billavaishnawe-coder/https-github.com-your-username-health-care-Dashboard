# 🏥 Healthcare Data Analysis Dashboard (Excel Project)

## 📌 Project Overview

This project focuses on analyzing healthcare data using Microsoft Excel to monitor clinical and operational performance metrics. The dashboard provides insights into patient visits, diagnoses, treatments, costs, and outcomes through interactive visualizations and KPI tracking.

The goal is to help healthcare professionals and administrators make data-driven decisions by identifying trends and evaluating treatment effectiveness.

---

## 🎯 Objectives

* Analyze patient healthcare records.
* Monitor key healthcare performance indicators.
* Evaluate treatment costs and patient outcomes.
* Create an interactive dashboard using Excel.
* Generate actionable insights through data visualization.

---

## 📊 Dataset Information

The dataset contains the following fields:

* Patient ID
* Visit Date
* Gender
* Age
* Diagnosis
* Treatment
* Cost
* Length of Stay
* Outcome

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Excel Formulas (AVERAGEIFS, COUNTIFS, COUNTIF, AVERAGE)

---

## 📈 Key Performance Indicators (KPIs)

* Total Patients
* Average Treatment Cost
* Average Length of Stay
* Recovered Patients
* Recovery Rate (%)
* Number of Surgeries

---

## 📉 Dashboard Features

### 1. Patient Distribution by Diagnosis

Visualizes the number of patients for each diagnosis category.

### 2. Average Cost by Treatment Type

Shows treatment expenses across different treatment methods.

### 3. Patient Outcome Analysis

Displays recovery, improvement, and stable outcome rates.

### 4. Interactive Filtering

Slicers allow users to filter data by:

* Gender
* Diagnosis
* Outcome

---

## 🔍 Key Insights

* Diabetes and Hypertension are among the most common diagnoses.
* Surgical treatments have higher average costs.
* Recovery rates provide insight into treatment effectiveness.
* Length of stay impacts overall healthcare expenses.
